# Simple Interest Calculator

This project is a simple interest calculator created as a final project for the Coursera Git and GitHub course.

## Description
The calculator computes simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Features
- Takes principal amount
- Takes rate of interest
- Takes time period
- Calculates simple interest

## Technologies Used
- Bash / Shell Script
- Git
- GitHub

## How to Run
1. Clone the repository
2. Run the script using bash
